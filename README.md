
## Technologies

- React (v16) + Redux (v3)
- Webpack 4 (production and development configurations)
- SCSS support 
- ES2017+

### Scripts
```bash
# run development mode
npm run dev

# run production mode
npm run build
npm start

# run prettier
npm run prettier

# run lint
npm run lint

# run on a different port
HTTP_PORT=3001 npm run dev
```
